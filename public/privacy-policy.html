プライバシーポリシー

Slope Link
最終更新日: 2026年1月5日

[野田あさひ]（以下「運営者」といいます）は、本アプリケーション「Slope Link」（以下「本アプリ」といいます）をご利用のお客様（以下「ユーザー」といいます）の個人情報およびプライバシーの保護を重要視しています。本プライバシーポリシーでは、本アプリにおいて、ユーザーの個人情報をどのように収集、利用、保護するかについて説明します。

1. 収集する情報

1.1 アカウント情報

本アプリを利用するために、以下の情報を収集します：

  • メールアドレス
  • パスワード（暗号化して保存）
  • ユーザーID（自動生成）

1.2 プロフィール情報

ユーザーが任意で提供する以下の情報を収集します：

  • 表示名
  • プロフィール画像（アバター）
  • ヘッダー画像
  • 国コード
  • 使用言語
  • スキルレベル
  • ライディングスタイル
  • 自己紹介文
  • ホームリゾート
  • ギア情報（ボード、バインディング、ブーツ、その他装備）

1.3 利用情報

本アプリの利用に伴い、以下の情報を収集します：

  • イベント情報：作成・参加したイベント、イベント申請、参加履歴
  • 投稿・コメント：Snowfeedへの投稿、コメント、いいね
  • チャットメッセージ：イベントチャット内のメッセージおよび画像
  • ソーシャル情報：★登録したユーザー、ブロックしたユーザー
  • 評価・レビュー：リゾート評価、スポットレビュー
  • 報告情報：不適切なコンテンツやユーザーの報告

1.4 画像・メディアファイル

以下の画像を収集・保存します：

  • プロフィール画像（アバター、ヘッダー）
  • イベント写真
  • Snowfeed投稿画像
  • チャットメッセージ内の画像

※ 画像は10MB以下、JPEG/PNG/WebP/GIF形式に制限されます。

1.5 位置情報

本アプリでは、スキーリゾートおよび関連施設（飲食店、温泉など）の位置情報を保存・利用します。ユーザーの現在位置をリアルタイムで収集することはありませんが、リゾートの検出および気象情報の取得のために位置情報を参照します。

1.6 デバイス情報

以下のデバイス情報を自動的に収集します：

  • デバイスの種類（iOS/Android）
  • OSバージョン
  • アプリバージョン
  • デバイスモデル
  • 言語設定
  • プッシュ通知トークン

1.7 分析・診断情報

アプリの品質向上のため、以下のツールを使用して情報を収集します：

  • Sentry：クラッシュレポート、エラー情報、デバッグログ
  • Amplitude：ユーザー行動分析（画面遷移、イベント操作など）
  • Firebase：アプリ分析、パフォーマンス監視

※ これらのツールは、パスワードやトークンなどの機密情報を収集しないよう設定されています。

2. 情報の利用目的

収集した情報は、以下の目的で利用します：

  1. サービス提供：アカウント管理、イベント参加、チャット、投稿機能の提供
  2. 機能改善：ユーザー体験の向上、新機能の開発
  3. コミュニケーション：プッシュ通知、重要なお知らせの送信
  4. セキュリティ：不正利用の防止、アカウントの保護
  5. 分析：利用統計の作成、アプリの最適化
  6. カスタマーサポート：お問い合わせ対応、トラブルシューティング
  7. 法令遵守：法的義務の履行

3. 情報の共有

3.1 公開情報

以下の情報は、他のユーザーに公開されます：

  • 表示名、プロフィール画像、自己紹介文
  • スキルレベル、ライディングスタイル、ギア情報
  • 作成したイベント、Snowfeed投稿
  • コメント、いいね
  • リゾート評価、スポットレビュー

3.2 第三者サービス

以下のサービスプロバイダーと情報を共有することがあります：

  • Supabase：データベース、認証、ストレージ管理
  • Sentry：エラー追跡とパフォーマンス監視
  • Amplitude：ユーザー行動分析
  • Firebase/Google：プッシュ通知、アプリ分析

これらのサービスは、それぞれのプライバシーポリシーに従って情報を管理します。

3.3 法的要請

法律、規制、法的手続き、または政府機関からの要請に応じて、ユーザー情報を開示することがあります。

4. データの保存期間

ユーザー情報は、以下の期間保存されます：

  • アカウント情報：アカウント削除まで
  • 投稿・コメント：削除されるまで、または法的要件に基づく期間
  • チャットメッセージ：削除されるまで
  • 分析データ：最大2年間
  • ログデータ：最大90日間

5. データの保護

運営者は、ユーザー情報を保護するため、以下のセキュリティ対策を実施しています：

  • SSL/TLS暗号化通信
  • パスワードの暗号化保存
  • アクセス制御とアクセス権限管理
  • 定期的なセキュリティ監査
  • データバックアップ

6. ユーザーの権利

ユーザーは、自身の個人情報に関して以下の権利を有します：

  • アクセス権：保存されている個人情報の確認
  • 訂正権：誤った情報の訂正
  • 削除権：個人情報の削除要請（アカウント削除）
  • データポータビリティ：データのエクスポート
  • 異議申し立て：情報処理方法への異議

6.1 アカウント削除

アカウントを削除すると、以下のデータがすべて削除されます：

  • プロフィール情報
  • 投稿・コメント・いいね
  • イベント参加履歴
  • チャットメッセージ
  • ★登録・ブロックリスト
  • 画像・メディアファイル
  • 認証情報

アカウント削除方法： こちらのページからアカウントを削除できます。

注意：アカウント削除は取り消せません。削除されたデータは復元できませんので、慎重にご検討ください。

7. プッシュ通知

本アプリでは、以下の通知を送信することがあります：

  • イベント申請の承認・却下通知
  • イベント開始リマインダー
  • イベントキャンセル通知
  • 新規参加者・申請通知（ホスト向け）
  • ★登録ユーザーのイベント参加通知
  • イベント後評価リマインダー
  • チャットメッセージ通知

通知設定は、アプリ内の「設定」→「通知設定」から変更できます。

8. Cookie・類似技術

本アプリでは、Cookieの代わりにローカルストレージ（AsyncStorage）を使用して、以下の情報を保存します：

  • セッション情報
  • 通知設定
  • 言語設定
  • オフラインキャッシュ

9. 子供のプライバシー

本アプリは、13歳未満のお子様を対象としていません。13歳未満のお子様の個人情報を故意に収集することはありません。万が一、13歳未満のお子様の情報が収集されていることが判明した場合、速やかに削除します。

10. 国際データ転送

本アプリは、Supabase（米国）およびその他のサービスプロバイダーを利用しており、ユーザー情報が国外に転送される場合があります。これらのサービスは、適切なセキュリティ対策を実施しています。

11. プライバシーポリシーの変更

運営者は、本プライバシーポリシーを随時更新することがあります。重要な変更がある場合は、アプリ内通知またはメールでお知らせします。最新版は常に本ページで確認できます。

Asahi Noda ("we," "us," or "our") values the privacy and personal information of users ("you" or "users") of our application "Slope Link" (the "App"). This Privacy Policy explains how we collect, use, and protect your personal information when you use the App.

1. Information We Collect

1.1 Account Information

To use the App, we collect the following information:

  • Email address
  • Password (encrypted and stored securely)
  • User ID (automatically generated)

1.2 Profile Information

Users may voluntarily provide the following information:

  • Display name
  • Profile picture (avatar)
  • Header image
  • Country code
  • Languages
  • Skill level
  • Riding styles
  • Bio
  • Home resort
  • Gear information (board, bindings, boots, other equipment)

1.3 Usage Information

While using the App, we collect the following information:

  • Event Information: Events created/joined, applications, participation history
  • Posts & Comments: Snowfeed posts, comments, likes
  • Chat Messages: Messages and images in event chats
  • Social Information: Starred users, blocked users
  • Ratings & Reviews: Resort ratings, spot reviews
  • Reports: Reports of inappropriate content or users

1.4 Images & Media Files

We collect and store the following images:

  • Profile images (avatar, header)
  • Event photos
  • Snowfeed post images
  • Chat message images

*Images are limited to 10MB and JPEG/PNG/WebP/GIF formats.

1.5 Location Information

The App stores and uses location information for ski resorts and related facilities (restaurants, hot springs, etc.). We do not collect users' real-time location, but we reference location data for resort detection and weather information.

1.6 Device Information

We automatically collect the following device information:

  • Device type (iOS/Android)
  • OS version
  • App version
  • Device model
  • Language settings
  • Push notification token

1.7 Analytics & Diagnostic Information

To improve app quality, we use the following tools to collect information:

  • Sentry: Crash reports, error information, debug logs
  • Amplitude: User behavior analysis (screen navigation, event interactions, etc.)
  • Firebase: App analytics, performance monitoring

*These tools are configured not to collect sensitive information such as passwords or tokens.

2. How We Use Your Information

We use collected information for the following purposes:

  1. Service Provision: Account management, event participation, chat, posting features
  2. Improvement: Enhancing user experience, developing new features
  3. Communication: Sending push notifications and important announcements
  4. Security: Preventing fraud, protecting accounts
  5. Analytics: Creating usage statistics, optimizing the app
  6. Customer Support: Responding to inquiries, troubleshooting
  7. Legal Compliance: Fulfilling legal obligations

The following information is shared with other users:

How to delete your account: You can delete your account from this page.
